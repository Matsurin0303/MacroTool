# プロダクト仕様書（MacroTool）

## 1. 目的
本ツールは、ユーザーの操作を記録し、編集し、保存したマクロを再生できる WinForms アプリケーションである。  
本仕様書は「ユーザー視点で何ができるか」と「本版での対象範囲」を定義する。

## 2. 前提
- 開発環境：Visual Studio 2026
- 言語：C#
- 設計思想：ドメイン駆動設計（DDD）＋ テスト駆動開発（TDD）
- UI仕様：`docs/images` を正として参照し、必要な操作・入力制約は文書で補完する

## 3. 対象範囲（スコープ）
### 3.1 本版（現行開発で扱う範囲）
- 仕様書（`MacroTool_MacroSpecification_v1.0.md`）に記載される機能のうち、
  - 「実装済」「一部実装済」「UIのみ実装」「未実装（ただし本版対象）」を対象とする
- マクロの「記録・編集・保存・再生」に関わる基本機能
- UI仕様（`docs/images`）に記載された画面構成の再現と、操作・入力制約の定義

### 3.2 対象外（将来実装予定）
- 仕様書に「将来予定」として示される機能
- 本版の実装範囲に含めない機能は、機能仕様（状態列）で **「対象外（将来予定）」** と明記する  
  ※対象外は実装しない（本プロジェクト方針）

## 4. ユースケース（代表）
1. 新規マクロを作成する  
2. 操作を記録する  
3. 記録結果を編集する（ステップの追加・削除・並べ替え等）  
4. マクロを保存／読み込みする  
5. マクロを再生し、必要に応じて停止・キャンセルする  
6. 再生中、実行中ステップがUI上で追従（自動選択）される

## 5. 用語（概要）
- Macro：マクロ全体
- Step：マクロ内の1手順
- Action：Stepが持つ具体的な操作（クリック、待機、判定など）
- Label：GoTo等で参照される一意なラベル
- GoTo：ターゲット（Start/Next/End/Label…）へ実行位置を移動する制御

詳細は [ドメイン仕様](./04_Domain/Domain_Model.md) を参照。

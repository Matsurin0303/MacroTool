# 非機能仕様書（MacroTool）

- Version: **Macro_v1.0.0**
- 更新日: 2026-03-18
- 対象: MacroTool（WinForms / C# / DDD + TDD）
- 参照: `docs/02_Requirements/Functional_Spec.md` / `docs/03_Architecture/Architecture_Overview.md` / `docs/03_Architecture/Error_Handling_Policy.md`

---

## 1. 目的

本書は、`Macro仕様書_v7.xlsx` を満たすために必要な**非機能上の要求と設計方針**を定義する。  
本版では、実装判断に直接必要な項目を優先して定義し、数値目標が未確定なものは **未確定事項** として明示する。

---

## 2. 適用範囲

### 2.1 本版で扱う非機能項目
- 保守性
- テスト容易性
- 障害解析容易性
- データ互換性
- 予測可能な再生動作
- ローカル実行前提の安全性

### 2.2 本版で扱わないもの
- サーバー運用前提の可用性要件
- クラウド連携前提のSLA
- 将来機能向けのAI / Network固有要件

---

## 3. 前提条件

- 開発環境は **Visual Studio 2026** とする。
- 実装言語は **C#** とする。
- UIは **WinForms** とする。
- 設計思想は **DDD + TDD** とする。
- 通常保存形式は **JSON**、交換形式は **CSV** とする。
- 単体で利用可能な **ローカルデスクトップアプリ** を前提とする。

---

## 4. 品質特性

### 4.1 保守性
- UI / Application / Domain / Infrastructure の責務を分離する。
- UIは Domain を直接操作しない。
- 仕様上の不変条件は Domain へ集約する。
- 外部依存（ファイルI/O、入力送信、画面取得）はインターフェース越しに扱う。

### 4.2 テスト容易性
- UseCase単位でテスト可能な構造とする。
- Domain ルールは UI から独立して検証可能であること。
- 再生制御、入力送信、画面取得、時刻取得は差し替え可能であること。
- 仕様の正は `Macro仕様書_v7.xlsx` と各 md 仕様書の整合で判断する。

### 4.3 予測可能性
- 再生状態の公開状態は **Idle / Playing** のみとする。
- 再生開始可否は開始前バリデーションで決定する。
- 停止要求はキャンセル可能境界で反映する。
- 永続化トークンは名称揺れを許容しない。

### 4.4 障害解析容易性
- 失敗は `DomainError / ApplicationError / SystemError` に分類する。
- 原因調査に必要なログを残せる構造とする。
- UI表示用メッセージと内部ログ情報を分離する。

### 4.5 データ互換性
- JSON と CSV の版は `Macro_vX.Y.Z` と関連仕様書で管理する。
- 本版対象外機能は永続化形式へ混入させない。
- CSVヘッダ、DTO、JSONトークンの語彙は関連仕様書で統一する。

### 4.6 ローカル実行安全性
- 外部プログラム実行、ファイル読込、画面取得は明示的なユーザー操作から開始されること。
- 予期しない自動送信・自動通信を本版前提にしない。
- 再生停止手段を UI / Hotkey / 入力検知設定から提供可能にする。

---

## 5. ログ・診断方針

- 例外は握り潰さず分類して扱う。
- 再生系は、少なくとも以下を識別できること。
  - 再生開始
  - 対象Macro
  - 実行Step
  - 終了理由
  - 失敗分類
- ログ出力先・保持期間・詳細レベルは本版未確定とする。

---

## 6. データ保全方針

- Save / Save As / Open / New / Recent Files は未保存確認の対象とする。
- 破損ファイル、形式不正、互換外データは ApplicationError として扱う。
- Import / Export では、内部保存形式(JSON)と交換形式(CSV)を混同しない。

---

## 7. UI/UX方針

- UIレイアウトの正は `docs/images` と `docs/05_UI/UI_Spec.md` とする。
- 本版対象外機能は、画像が存在しても仕様対象に含めない。
- エラーメッセージはユーザーが次に取るべき行動を判断できる粒度とする。

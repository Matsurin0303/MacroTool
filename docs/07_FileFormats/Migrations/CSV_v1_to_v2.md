# CSVマイグレーション仕様：CSV_v1.0 → CSV_v2.0（将来予定）

## 1. 本ドキュメントの位置付け

本ドキュメントは、**将来 `CSV_v2.0` を導入する場合の検討用草案**である。

**Macro_v1.0.0 本版では実装対象外**であり、Import 時に `CSV_v1.0` から `CSV_v2.0` への自動変換は行わない。

## 2. 本版との関係

Macro_v1.0.0 本版では、以下を正式仕様とする。

- Import 対応: `CSV_v1.0` のみ
- Export 対応: `CSV_v1.0` のみ
- スキーマ識別: ヘッダ完全一致
- メタ行方式: 未採用
- 自動マイグレーション: 未実装

そのため、本書の内容は **実装義務を持たない参考資料** とする。

## 3. 将来導入時の前提条件

本書を正式仕様へ昇格するには、少なくとも以下を別途確定する必要がある。

- `CSV_v2.0` の正式ヘッダ
- `CSV_v2.0` の Export 仕様
- `#SchemaVersion` などメタ行方式の正式採用
- `Version_Mapping_Table.md` の更新
- 自動変換失敗時のエラー / 警告ポリシー
- 旧CSVのサポート期限

## 4. 参考として残す内容

以下の観点は、将来の設計検討時に参照してよい。

- `v1 -> v2` の段階的変換という考え方
- 値の正規化（enum / bool / GoTo表記揺れ吸収）
- メタ行導入の方向性
- 変換不能条件の洗い出し観点

ただし、本版では **いずれも未採用** である。

## 5. 現時点での扱い

- 実装タスクは起票しない
- テスト対象に含めない
- `CSV_v2.0` を前提とした UI / Import / Export 実装を行わない

## 6. 正式化のタイミング

次に該当するバージョン改訂時に、必要であれば本書を見直して正式仕様へ格上げする。

- `CSV_v2.0` を本版へ導入すると判断した時点
- `Macro_v2.x` 以降で互換移行を実装対象に含める時点
